Wildlife doesn't respect borders.
Neither does this map.

SpeciesMap is an open platform that brings together sighting records, GPS tracking data, and conservation assessments from the world's leading scientific databases — in one place, with one interface.

We don't collect data. We connect it. Every sighting you see is sourced from GBIF, Movebank, eBird, or the IUCN Red List — then normalised, mapped, and made navigable for anyone from a curious school student to a field researcher.

This is a passion project, built in the open. It started as a curiosity about where the data actually is — and became a mission to make it visible.

How it works

01 — Collect
Data from the source

We pull sighting records, GPS collar data, and range polygons directly from GBIF, Movebank, eBird, and the IUCN Red List. Updated regularly.

02 — Normalise
One schema, all species

Each source formats data differently. We map everything into a unified PostGIS schema — species, location, date, class, conservation status — so it's queryable as one.

03 — Visualise
Maps that move

Interactive terrain maps with animated migration routes, species range overlays, seasonal activity charts, and IUCN conservation status — built for exploration.
